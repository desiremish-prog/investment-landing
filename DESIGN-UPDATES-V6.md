# Обновление дизайна лендинга - Версия 6

## Выполненные изменения согласно ТЗ

### ✅ 1. Цветовая схема
- **Градиентные фоны**: Внедрены современные градиенты с синими, зелёными и графитовыми оттенками
- **Основная палитра**:
  - Primary Blue: `#1e40af` → `#1e3a8a`
  - Secondary Green: `#10b981`
  - Accent Teal: `#14b8a6`
  - Graphite: `#374151`
- **Градиент героя**: `linear-gradient(135deg, #1e3a8a, #1e40af, #14b8a6)`
- **Фон страницы**: Плавный градиент от светло-голубого к белому

### ✅ 2. Анимации
#### Появление блоков при скролле:
- Fade-in эффект для всех карточек
- Staggered анимация (каждая карточка появляется с задержкой)
- Использован Intersection Observer API для производительности

#### Hover-эффекты:
- **Кнопки**: Scale + translateY с градиентной подсветкой
- **Карточки**: Плавный подъём, изменение бордера и тени
- **Иконки**: Вращение и масштабирование (360° для круглых иконок)
- **CTA кнопки**: Эффект "волны света" через псевдоэлемент ::before

### ✅ 3. Карточный дизайн
- **Увеличен border-radius**: 12px → 16px для всех карточек
- **Улучшены тени**: Добавлены цветные тени с эффектом глубины
- **Градиентные акценты**: При hover карточки получают градиентный фон
- **Единообразие**: Все карточки (проблемы, услуги, кейсы, доверие) имеют одинаковый стиль

### ✅ 4. Визуальные элементы
#### Инфографика:
- Градиентные иконки с анимацией вращения
- Цветные числовые показатели с gradient text
- Улучшенные визуальные разделители между секциями

#### Визуальные разделители:
- Тонкие линии между секциями
- Градиентные подчёркивания под заголовками
- Цветные акценты на левых границах карточек

### ✅ 5. Типографика
- **Шрифт**: Inter (современный sans-serif)
- **Улучшена иерархия**: Чёткие размеры для H1, H2, H3
- **Text shadow**: Добавлены для героя и CTA секций
- **Letter-spacing**: Оптимизирован для читаемости

### ✅ 6. Кнопки (CTA)
- **Градиентный фон**: Green → Teal
- **Увеличенный radius**: 12px
- **Анимация свечения**: При hover тень усиливается
- **Эффект волны**: Анимированный градиент проходит по кнопке
- **Единообразие**: Все CTA кнопки имеют одинаковый стиль

### ✅ 7. Адаптивность
- Улучшена типографика для мобильных
- Оптимизированы размеры кнопок на малых экранах
- Правильное поведение анимаций на touch-устройствах
- Сохранены все responsive breakpoints

## Технические детали

### CSS переменные (обновлены):
```css
--primary-color: #1e40af
--secondary-color: #10b981
--accent-teal: #14b8a6
--shadow-colored: 0 10px 30px -5px rgba(30, 64, 175, 0.2)
```

### JavaScript улучшения:
- Добавлена функция `initScrollAnimations()`
- Использован IntersectionObserver для производительности
- Настроены пороги видимости и rootMargin

### Ключевые CSS классы:
- `.fade-in`, `.slide-in-left`, `.slide-in-right` - базовые анимации
- `.visible` - триггер для активации анимаций
- Stagger delays для последовательного появления элементов
