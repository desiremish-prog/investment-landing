QUICK START:
------------

VIEW DEMO:
  1. Open "index.html" in your browser
  2. Click "Record for Meeting" button
  3. Test form validation and submission

INSTALL ON WORDPRESS:
  1. Copy "wordpress-theme/" to "wp-content/themes/investment-landing"
  2. Copy "css/style.css" to "assets/css/style.css"
  3. Activate theme in WordPress admin
  4. Check admin menu "Requests" section
  5. Test form submission

DEPLOY STATIC VERSION:
  1. Upload index.html, css/, js/, privacy-policy.html to hosting
  2. Configure SSL certificate
  3. Update email addresses
  4. Test functionality

WORDPRESS DATABASE:
-------------------

Table: wp_consultation_requests
Columns:
  - id (bigint, primary key)
  - name (varchar 200)
  - email (varchar 200)
  - phone (varchar 50)
  - comment (text)
  - ip_address (varchar 100)
  - user_agent (text)
  - created_at (datetime)
  - status (varchar 50, default 'new')
